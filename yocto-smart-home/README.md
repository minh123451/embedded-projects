# Embedded Smart Home System (Yocto Linux)

Embedded smart home control system built on Raspberry Pi 4 using a custom Yocto Linux OS with self-developed GPIO and I2C drivers and a Qt5 GUI application.

## Features
- Custom Yocto Linux image for Raspberry Pi 4
- Custom GPIO driver implemented as Yocto meta-layer
- Custom I2C driver for Raspberry Pi – Arduino communication
- Qt5-based GUI application
- LED control and switch input handling

## System Overview
- Raspberry Pi 4 runs Yocto Linux and Qt5 GUI
- GPIO and I2C implemented as custom Yocto meta-layers
- Arduino communicates with Raspberry Pi via I2C
- Qt GUI controls hardware devices through drivers

## Technologies
Yocto Project, Embedded Linux, Qt5, Raspberry Pi 4, GPIO Driver, I2C Driver, Arduino
