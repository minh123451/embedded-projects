# Raspberry Pi AI Security Camera

Mobile AI-based security camera system using Raspberry Pi.

## Features
- Real-time video streaming
- AI object detection
- Client-server architecture
- Web-based monitoring interface
- GPIO alarm control

## Files
- client.py: Raspberry Pi client
- server.py: AI processing server
- giaodien.html: Web UI
