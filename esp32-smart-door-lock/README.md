# ESP32 Smart Door Lock System

Smart door lock system using ESP32 with multi-authentication, IoT remote control, and smoke alarm.

## Features
- PIN code access (keypad)
- RFID card authentication
- Fingerprint authentication
- Remote door control via Blynk
- Servo-based door lock
- Smoke detection and buzzer alarm
- LCD status display
